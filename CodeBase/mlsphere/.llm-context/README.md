# MLSphere Project - LLM Context Documentation

## Purpose of This Folder

This folder contains documentation specifically written for Large Language Models (LLMs) and AI assistants to understand the MLSphere project architecture, codebase structure, and design decisions.

## Files in This Folder

- `PROJECT_OVERVIEW.md` (✅ Complete, 15KB) - High-level project description, architecture, and workflows
- `ARCHITECTURE.md` (✅ Complete, 18KB) - System architecture, components, and interactions
- `CODEBASE_MAP.md` (✅ Complete, 19KB) - File structure, responsibilities, and data flows
- `DESIGN_DECISIONS.md` (✅ Complete, 16KB) - Key technical decisions with rationale and trade-offs
- `API_CONTRACTS.md` (✅ Complete, 14KB) - API specifications and interface contracts
- `DEVELOPMENT_GUIDE.md` (✅ Complete, 13KB) - Developer guide for extending the system

**Total Documentation**: ~95KB of comprehensive technical context

## How to Use

When working with an LLM/AI assistant on this project:

1. **Start with PROJECT_OVERVIEW.md** to understand the big picture
2. **Reference ARCHITECTURE.md** for system design and component interactions
3. **Use CODEBASE_MAP.md** to locate specific files, functions, and understand code structure
4. **Consult DESIGN_DECISIONS.md** to understand "why" certain approaches were taken
5. **Check API_CONTRACTS.md** for detailed interface specifications
6. **Follow DEVELOPMENT_GUIDE.md** when adding features or making changes
